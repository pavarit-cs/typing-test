# Typing Test CLI — Project Plan

## Project Goal
สร้างโปรแกรม CLI ให้ผู้ใช้ทดสอบความเร็วและความแม่นยำในการพิมพ์  
โดยจะแสดงข้อความสุ่มให้ผู้ใช้พิมพ์ จากนั้นคำนวณเวลา, WPM, และเปอร์เซ็นต์ความถูกต้อง
